## Overview

AI Quiz Generator is an interactive web application designed to help users create, practice, and test their knowledge through dynamic quizzes generated by AI. Whether you are a student, professional, or lifelong learner, this tool provides a personalized and engaging way to learn new topics, improve memory, and enhance your knowledge retention

The application leverages modern frontend technologies to deliver a smooth, responsive, and intuitive user experience. Users can generate multiple-choice quizzes instantly, customize the quiz according to their preferences, and get immediate feedback on their answers.

## Key Features

* **Dynamic Quiz Generation**: Generate quizzes automatically based on user-selected topics, difficulty levels, and number of questions.
* **Customizable Options**: Choose from different subjects, adjust difficulty, and select the number of questions for a personalized learning experience.
* **Multiple-Choice Questions**: Each quiz is composed of multiple-choice questions for effective assessment.
* **Instant Feedback**: Users receive immediate feedback after submitting each answer or completing the quiz.
* **Responsive Design**: Optimized for desktops, tablets, and mobile devices for learning on the go.
* **Fast & Lightweight**: Minimal load times ensure a smooth and responsive user experience.
* **Results Overview**: View your performance after completing a quiz to track improvement and identify areas of strength and weakness.

## Technology Stack

The project is built with modern web development tools and frameworks:

* **Frontend**: React, TypeScript, Vite
* **Styling**: Tailwind CSS
* **AI-Powered**: Quiz generation is powered by advanced AI techniques for unique and diverse question sets
* **Deployment & Hosting**: Hosted and deployed on Vercel for instant access worldwide

## Getting Started

### Prerequisites

* Web browser (Chrome, Firefox, Edge, Safari, etc.)
* Internet connection for AI quiz generation and live demo

### Installation (Local Setup)

1. **Clone the repository**

   ```bash
   git clone <your-repository-url>
   cd <repository-folder>
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Start the development server**

   ```bash
   npm run dev
   ```

   Open the provided link in your browser to access the app locally.

4. **Build for production**

   ```bash
   npm run build
   ```

   This creates a production-ready version of the app that can be deployed or shared.

## Usage

1. Open the **live demo** or your local instance.
2. Select a **topic** you want to learn or practice.
3. Choose the **difficulty level** (easy, medium, hard).
4. Select the **number of questions** you want in the quiz.
5. Click **Generate Quiz**.
6. Answer the questions. The app provides **instant feedback** for each answer.
7. Review your **final score** and **correct answers** at the end.
8. Repeat with different topics or difficulty levels for continuous learning.
